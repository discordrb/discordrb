Overview

Voice regions are the locations of servers that handle voice communication in Discord

Instance Attribute Summary (collapse)

  • - (true, false) custom readonly

    Whether this is a custom voice region (used for events/etc).
  • - (true, false) deprecated readonly

    Whether this is a deprecated voice region (avoid switching to these).
  • - (String) id (also: #to_s) readonly

    Unique ID for the region.
  • - (String) name readonly

    Name of the region.
  • - (true, false) optimal readonly

    If this voice server is the closest to the client.
  • - (String) sample_hostname readonly

    An example hostname for the region.
  • - (Integer) sample_port readonly

    An example port for the region.
  • - (true, false) vip readonly
